# 🎵 Proyecto Semana 04 – Generador de Mensajes de Dominio

## Dominio: App de Música y Podcast

### 🎯 Objetivo

Crear un generador de mensajes en consola utilizando **métodos de string en JavaScript** y **template literals**.
El programa simula información de contenido dentro de una **App de Música y Podcast**, como un episodio o canción disponible en la plataforma.

---

## ⚙️ Tecnologías usadas

* JavaScript
* Node.js
* Visual Studio Code

---

## 📂 Estructura del proyecto

```
3-proyecto/
│
├── README.md
│
└── starter/
    └── script.js
```

---

## 📊 Funcionalidad

El script genera dos tipos de mensajes:

### 1️⃣ Ficha del contenido

Muestra información del contenido de la app:

* Nombre del contenido
* Categoría
* Código
* Descripción
* Validaciones

---

### 2️⃣ Mensaje de notificación

Genera una alerta corta indicando que hay nuevo contenido disponible en la aplicación.

Ejemplo:

```
📢 Nuevo contenido disponible: Sonidos del Universo (POD-101)
```

---

## 🧠 Métodos de string utilizados

El proyecto utiliza varios métodos de string:

```
trim()
toUpperCase()
includes()
startsWith()
slice()
replace()
```

Estos métodos permiten limpiar, transformar y analizar texto.
