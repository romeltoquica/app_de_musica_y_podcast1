# 🎧 Proyecto Semana 1: Ficha de Presentación de Dominio

## 📋 Dominio Asignado

**Dominio:** App de Música y Podcasts

Este proyecto consiste en crear un **script en JavaScript** que muestre en la consola una ficha de información sobre una entidad del dominio **aplicaciones de música y podcasts**, como una playlist, un podcast o un episodio.

---

## 🎯 Objetivo

El objetivo es practicar los conceptos básicos vistos en la semana:

* Uso de `console.log()`
* Tipos de datos primitivos (`string`, `number`, `boolean`)
* Expresiones aritméticas
* Uso de comentarios en el código

El script debe mostrar una **ficha organizada en consola** con información sobre una entidad del dominio.

---

## 📋 Descripción del Proyecto

La entidad elegida para la ficha es una **playlist de música** dentro de una aplicación de música y podcasts.

El programa muestra información como:

* Nombre de la playlist
* Código identificador
* Categoría
* Número de canciones
* Duración total
* Estados de disponibilidad
* Un cálculo numérico (por ejemplo precio con descuento)

Toda la información se imprime en la consola usando **`console.log()`**.

## ▶️ Cómo ejecutar el proyecto

1. Abre la terminal.
2. Ve a la carpeta del proyecto.
3. Ejecuta el siguiente comando:

```bash
node starter/script.js
```

Si todo está correcto, se mostrará la **ficha de la playlist en la consola**.
