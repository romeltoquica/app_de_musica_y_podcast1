# 🎵 Proyecto Semana 03 — Calculadora de Dominio

## Dominio: App de Música y Podcast

### 🎯 Objetivo
Este proyecto consiste en crear una **calculadora aplicada a una App de Música y Podcast**, utilizando los operadores aprendidos en JavaScript durante la semana.

La aplicación simula operaciones comunes dentro de una plataforma de streaming como:

- cálculo de ingresos por suscripciones
- total de usuarios
- promedio de reproducciones
- validación de usuarios premium
- control de almacenamiento

---

# ⚙️ Tecnologías utilizadas

- JavaScript
- Node.js
- Visual Studio Code

---

# 📊 Funcionalidades

### 1. Cálculo de ingresos
Se calcula el ingreso mensual generado por los usuarios premium.

### 2. Total de usuarios
Se suman los usuarios premium y gratuitos para conocer el total de usuarios de la app.

### 3. Promedio de reproducciones
Se calcula el promedio de reproducciones por usuario.

### 4. Actualización de usuarios
Se utilizan operadores de **asignación compuesta** para actualizar la cantidad de usuarios.
